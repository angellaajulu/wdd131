Home Gardening – Website Planning Document

Site Name

Home Gardening Hub

This name clearly communicates that the site provides simple, friendly gardening guidance for beginners.

Site Purpose

The purpose of this website is to teach beginners how to start and care for a home garden. It will include plant care guides, watering instructions, soil preparation tips, and quick solutions for common issues like yellow leaves or pests.

Scenarios

  • What are some easy plants beginners can grow indoors or outdoors?
  • How can a new gardener fix common plant issues like overwatering and yellowing leaves?

Color Schema

Primary Color – #2d6a4f

Used for headings, borders, accents, and section titles.

Secondary Color – #b7e4c7

Used for the page background and section contrast.

Typography

Poppins – Used for headings and strong labels.

Open Sans – Used for all body text and lists for readability.
